# Trader Performance vs Market Sentiment Analysis

## Project Overview
This project analyzes how market sentiment (Fear vs Greed) impacts trader behavior and performance using Hyperliquid trading data.

---

## Dataset
- Bitcoin Fear & Greed Index
- Historical Trader Data

---

## Methodology
- Data cleaning and preprocessing
- Date alignment and merging datasets
- Feature engineering:
  - Daily PnL
  - Win rate
  - Average trade size
  - Leverage
  - Trades per day
  - Long/Short ratio
- Exploratory Data Analysis (EDA)
- Trader segmentation

---

## 📈 Key Insights
- Trader performance varies across sentiment regimes
- Higher trading activity observed during Greed periods
- Risk-taking behavior increases with positive sentiment

---

## Strategy Recommendations
- Reduce leverage during Fear periods
- Increase trade activity selectively during Greed periods

---

1. Open notebook in Google Colab
2. Upload datasets
